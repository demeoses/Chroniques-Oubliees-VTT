# Chroniques Oubliées - Table Virtuelle D&D

Version statique compatible GitHub Pages.

## Fichiers principaux
- index.html
- style.css
- script.js

## Mise en ligne
1. Créer un dépôt GitHub.
2. Envoyer tous les fichiers.
3. Aller dans Settings > Pages.
4. Choisir la branche main et le dossier root.

## Ressources
- assets/maps : cartes
- assets/tokens : pions
- assets/icons : icônes
